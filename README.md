# first bitcoin wallet

## Description

created a simple wallet using the bitcoin lib using P2PKH.

### Requirements

-   [Node.js](https://nodejs.org/en/) v20.0.0 or higher
-   [NPM](https://www.npmjs.com/) v9.0.0 or higher

#### Installation

```bash
$ npm install

```
#### Build the app

```bash
$ npm run build
```

#### Running the app

```bash
$ node index.js
```
